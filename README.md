
## PawPrint_Server
반려동물 추억 기록 및 펫로스 증후군 완화서비스 PawPrint의 서버입니다. <br>

## 기술 스텍

- 언어 : java
- 프레임워크 : Spring Boot
- 버전 컨트롤러 : git, github
- 배포 : Docker, AWS - EC2, S3
- DB : Mysql, AWS - RDS
